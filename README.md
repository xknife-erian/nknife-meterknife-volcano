# MeterKnife Lite 1.20

- 本项目是一个开源软件项目。
- 本项目开源协议是《木兰宽松许可证, 第1版》


## 开发历史

### 2020-1-14
- 历时5年了，一直忙于工作，久久没有再更新软件，在群里见非常多的朋友都在使用这个软件，非常高兴，但也是非常遗憾有不少功能没有实现。
- 近几日，将时间规划了一下，决定今年认真升级MeterKnife平台。
  1. MeterKnife Lite版在Gitee上保持开源，主要实现的是实现仪器数据的采集；
  2. MeterKnife Lite仍采用.Net Framework4.0做为开发平台，以适应XP操作系统；
  3. MeterKnife Professional（简称 MeterKnife）将定义为一个自动化工业采集平台，仪器采集将是其重要的一部份，但会更加深入的面向各种通讯手段的数据采集，例如485总线、MBUS总结、RTU/DTU、Socket主从、自定义设备等等……。作者希望平台能为设备、自控集成、MES实施和最终用户提供实时数据采集平台、生产监控平台、生产过程数据存储平台。
  4. MeterKnife Professional 有想法实现一套基于Web的组态，但是想法还不成熟，待定；

### 2015.11.28

- 基本功能已经实现。

### 2015.7.15
